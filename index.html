About

"ApodS" is a React Native mobile application made for every people. Our application offers unforgettable images of sky. There are two different ways to login. Thanks to that we can add Apods to observe panel and see it in future or show it to friends. We can also like Apods and look for them in ranking. There is also option to add comment or view comments of other users. Thanks to firebase database application work smooth. All this is just a beggining. As a developers we can assure that this project is going to be continued and become even greater.
